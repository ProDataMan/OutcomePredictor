# Privacy Policy for NFL Outcome Predictor

**Last Updated:** December 2025

## Introduction

NFL Outcome Predictor ("the App") is committed to protecting user privacy. This privacy policy explains our data practices.

## Data Collection

**We do not collect, store, or share any personal information.**

The App operates entirely using publicly available data sources and does not require user accounts, login credentials, or personal information of any kind.

## Third-Party Data Sources

The App retrieves data from the following public sources:

### ESPN API
- **Purpose:** NFL team information, game schedules, and scores
- **Data Type:** Publicly available NFL game data
- **Privacy Policy:** https://privacy.thewaltdisneycompany.com/en/

### The Odds API
- **Purpose:** Betting odds and lines for informational purposes
- **Data Type:** Publicly available betting market data
- **Privacy Policy:** https://the-odds-api.com/privacy

### Microsoft Azure
- **Purpose:** Application server hosting
- **Data Type:** No user data is stored on our servers
- **Privacy Policy:** https://privacy.microsoft.com/

## Internet Access

The App requires an active internet connection to:
- Fetch current NFL schedules and scores from ESPN
- Retrieve betting odds from The Odds API
- Generate game predictions from our prediction server

All network requests are made directly from your device to the respective services. We do not intercept, log, or store any of these requests.

## Analytics and Tracking

**We do not use analytics, tracking, or advertising services.**

No user behavior, app usage, or device information is collected or transmitted to any analytics platforms.

## Location Data

The App does not access, collect, or use device location data.

## Device Permissions

The App only requires:
- **Internet Access:** To fetch game data and predictions

No other device permissions are requested or used.

## Data Storage

**No user data is stored on your device or our servers.**

The App displays data retrieved in real-time from public APIs. When you close the app, no information persists.

## Children's Privacy

The App is rated 4+ and is safe for all ages. We do not knowingly collect any information from children or any other users.

## Disclaimer

**The App is not affiliated with, endorsed by, or officially connected with:**
